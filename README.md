# Développement Personnel

Application Vite + React autonome pour explorer les piliers de développement personnel: sommeil, alimentation, productivité, confiance, finances, influence, habitudes et emploi du temps.

## Lancer le projet

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run build
npm run preview
npm run lint
```

Le projet ne demande pas de variables d'environnement pour démarrer en local.
